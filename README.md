# Astro + Tailwind Taskbar Starter

Minimal Astro site scaffolded with Tailwind CSS. Includes a drop-down taskbar, a hero section, and a CTA button that navigates to an action page.

Quick start (Windows):

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Dev server (default): http://localhost:4321/

Build:

```bash
npm run build
npm run preview
```
